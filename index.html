GeoMapFish Executive Summary

The GeoMapFish application allows to build rich and extensible WebGIS in an easy and flexible way. It is composed of a desktop WebGIS interface, an administration interface, an API for map integration in thirdparty websites and a mobile version.

Besides the OGC-Standard web services, a MapFish protocol adapted to the efficient communication between Client and Server is available. On this basis, complex and high performance web mapping applications can be built.

GeoMapFish combines some of the best Open Source tools in one application:

  • OpenLayers, ExtJS and GeoExt on the client side
  • Python modules (especially Papyrus based on Pyramid) on the server side
  • MapFish Print, a Java servlet dedicated to print geographic documents

GeoMapFish in production

Following online WebGIS are powered by GeoMapFish:

  • geoview.bl.ch - Geoview Basel Landschaft
  • map.cartoriviera.ch - Géoportail cartographique de la Riviera
  • mapfish.jura.ch - Géoportail du Système d'Information du Territoire Jurassien
  • map.geoplanet.ch - Geoplanet, guichet cartographique cantonal de l'Etat de Vaud
  • map.morges.ch - sitecom Système d'Information du Territoire de la Commune de Morges
  • sitn.ne.ch - Géoportail du Système d'Information du Territoire Neuchâtelois
  • map.nyon.ch - SITNyon Géoportail de la Ville de Nyon
  • map.regiogis-beo.ch - RegioGIS Berner Oberland
  • map.geo.sz.ch - webGIS Kanton Schwyz
  • sigip.ch - Système d'Information Géographique Intercommunal de Pully, Paudex et Belmont
  • geo.yverdon.ch - Système d'Information du Territoire Yverdon-les-Bains
